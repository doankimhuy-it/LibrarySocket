# LibrarySocket

## Cấu trúc message:
 - Đăng nhập: ...
 - Đăng ký: ...
 - Tra cứu: ...
 - Xem sách: ...
 - Tải sách: ...
 - Thoát: ...

## Cấu trúc file
 ### Server
 - 1 file dùng để khởi tạo kết nối
 - 1 file dùng để tạo GUI chính
 - Các chức năng
 - 1 folder dùng để lưu cơ sở dữ liệu

 ### Client
 - 1 file dùng để khởi tạo kết nối
 - 1 file dùng để tạo GUI chính
 - Các chức năng
 - 1 folder dùng để lưu sách

## Các chức năng đã làm được ở câu lý thuyết:
 - Kết nối.
 - Quản lý kết nối.
 - Thoát.
 - Giao diện.
